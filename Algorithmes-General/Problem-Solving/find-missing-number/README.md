## Find Missing Number

### Description

You are given a list containing numbers from 0 to n, but one number is missing.
Find the missing number.

### Rules

- The list contains n numbers.
- All numbers are unique.
- Numbers range from 0 to n.

### Input

A list of integers

### Output

The missing number

### Example

Input: `[3, 0, 1]`

Output: `2`
