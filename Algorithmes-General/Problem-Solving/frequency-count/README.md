## Frequency Count

### Description

Count how many times each element appears in a list.

### Rules

- The order does not matter.
- Each unique element should have a count.

### Input

A list of elements (numbers or strings)

### Output

A structure showing each element and its frequency

### Example

Input: `[1, 2, 2, 3, 3, 3]`

Output:

```
1 → 1
2 → 2
3 → 3
```
