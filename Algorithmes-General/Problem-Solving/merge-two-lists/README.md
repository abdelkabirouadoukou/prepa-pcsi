## Merge Two Lists

### Description

Given two sorted lists, merge them into one sorted list.

### Rules

- Both input lists are already sorted.
- The final list must also be sorted.
- All elements from both lists must appear in the result.

### Input

Two sorted lists of integers

### Output

One sorted list containing all elements

### Example

Input: `[1, 3, 5]` and `[2, 4, 6]`

Output: `[1, 2, 3, 4, 5, 6]`
