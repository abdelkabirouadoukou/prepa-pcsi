## Move Zeros

### Description

Move all 0 values to the end of the list while keeping the relative order of non-zero elements.

### Rules

- Do not change the order of non-zero elements.
- Only zeros should be moved.

### Input

A list of integers

### Output

The modified list

### Example

Input: `[0, 1, 0, 3, 12]`

Output: `[1, 3, 12, 0, 0]`
