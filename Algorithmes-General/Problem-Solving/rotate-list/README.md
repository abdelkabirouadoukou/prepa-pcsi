## Rotate List (Right Rotation)

### Description

Rotate a list to the right by k positions.

### Rules

- Rotation means elements at the end move to the front.
- k can be greater than the length of the list.
- The order of elements must be preserved after rotation.

### Input

- A list of integers
- An integer k (number of rotations)

### Output

The rotated list

### Example

Input: `list = [1, 2, 3, 4, 5], k = 2`

Output: `[4, 5, 1, 2, 3]`
