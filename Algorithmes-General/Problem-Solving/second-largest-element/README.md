## Second Largest Element

### Description

Given a list of integers, find the second largest number in the list.

### Rules

- The list contains at least two numbers.
- The second largest number must be different from the largest.
- If the largest number appears multiple times, it is still counted once.

### Input

A list of integers.

### Output

A single integer representing the second largest value.

### Example

Input: `[10, 5, 20, 8]`

Output: `10`
