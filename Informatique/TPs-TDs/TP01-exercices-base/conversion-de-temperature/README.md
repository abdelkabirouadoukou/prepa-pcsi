# Conversion de Température

## 📚 Description
Exercice d'introduction à la programmation Python pour convertir entre différentes unités de température.

## 🎯 Objectifs
- Maîtriser les variables et les types de données
- Effectuer des opérations arithmétiques simples
- Implémenter des formules mathématiques
- Utiliser les fonctions pour réutiliser du code

## 📝 Formules

### Celsius ↔ Fahrenheit
- C → F: `F = C × 9/5 + 32`
- F → C: `C = (F - 32) × 5/9`

### Celsius ↔ Kelvin
- C → K: `K = C + 273.15`
- K → C: `C = K - 273.15`
