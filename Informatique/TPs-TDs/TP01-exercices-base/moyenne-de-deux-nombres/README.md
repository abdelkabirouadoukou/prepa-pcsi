# Moyenne de Deux Nombres

## 📚 Description
Exercice simple pour calculer la moyenne arithmétique de deux nombres saisis par l'utilisateur.

## 🎯 Objectifs
- Pratiquer les opérations arithmétiques de base
- Gérer l'entrée utilisateur
- Afficher les résultats formatés
- Comprendre les types numériques (int, float)

## 📐 Formule
```
Moyenne = (nombre1 + nombre2) / 2
```

## 🔧 Extensions possibles
- Moyenne de 3, 4, ... n nombres
- Utiliser `sum()` et `len()` pour généraliser
- Moyenne pondérée
